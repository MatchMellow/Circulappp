# CirculAPP 

Aplicativo Android desenvolvido em Kotlin para facilitar a mobilidade no campus da UFRPE,
permitindo comunicação entre alunos e motoristas.

## Tecnologias
- Kotlin
- Jetpack Compose
- Firebase Realtime Database
- Google Maps SDK
